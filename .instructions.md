# ContaEC Development Guidelines

## Always Active Skills

### token-efficiency
**PERMANENTLY ACTIVE** - Apply on every tool call, file read, and command execution.

Minimize token waste in all operations:
- Avoid dumping entire files; use structured queries (`jq`, `yq`, `grep`)
- Filter output at the source rather than processing afterwards
- Use `head`, `cut`, `awk` for data projection
- Remember: each byte of output costs tokens and context
